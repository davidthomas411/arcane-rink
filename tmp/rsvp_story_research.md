# RSVP Story Presentation Notes

## Why this format

Rapid Serial Visual Presentation (RSVP) can keep momentum in short game interstitials because it reduces eye travel, but comprehension drops if pacing is too aggressive or if players cannot pause/review.

## Research-informed defaults used in this build

- Base speed: `250 WPM` (buttons exposed: `200 / 250 / 300`)
- Punctuation-aware timing:
  - comma/semicolon pauses are longer
  - sentence-ending punctuation pauses are longest
- User control is always available:
  - pause/resume
  - replay current panel
  - fast-forward
  - skip cutscene
- Text remains short and direct (single-line story beats per panel).
